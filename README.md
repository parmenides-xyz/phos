# Homer (helios-exex)

Light client for DATA Network (formerly Story), written in Rust. Homer converts an untrusted RPC endpoint into a safe unmanipulable local RPC for users. It syncs in seconds, requires no storage, and is lightweight enough to run in browser + on mobile devices.

Homer is not a fork, but extends upstream [Helios](https://github.com/a16z/helios), adding DATA-specific consensus (CometBFT) and a custom EVM.
