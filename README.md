# helios-exex (Homer)

Light client for DATA Network (formerly Story) in Rust, able to run natively, in browser, and on mobile devices.

Homer converts an untrusted RPC endpoint into a safe unmanipulable local RPC for users.

Homer is not a fork, but extends upstream [Helios](https://github.com/a16z/helios), adding DATA-specific consensus (CometBFT) and a custom EVM.
