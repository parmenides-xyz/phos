# DATA Network Protobuf Types

Generated Rust types for the DATA Network consensus adapter.

Vendored sources:

- `story/evmengine/v1/types/tx.proto`: `piplabs/story` commit
  `11a7f60a836233c9e9e546cad95958e1a3925f52`.
- Cosmos SDK protos: `piplabs/cosmos-sdk` module
  `v0.50.14-piplabs-v1.1`, as selected by the pinned DATA revision.
- `gogoproto/gogo.proto`: `cosmos/gogoproto` module `v1.7.0`.
- `cosmos_proto/cosmos.proto`: `cosmos/cosmos-proto` module
  `v1.0.0-beta.5`.

The legacy `story.evmengine.v1.types` package is retained because it is part of
the chain's protobuf wire format.
